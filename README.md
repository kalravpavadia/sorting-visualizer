# Sort visualization
Sorting algorithms OpenGl/GLut based visualizer.

## Features made
* Simple sorting algorithm customizable visualization engine. 
* 4 sorting algorithms for example (bubble, merge, heap, quick).
* 2D raster text UI.
* Keyboard driven control.
* Toggle render speed.

## Tools used:
* C++, STL.
* OpenGL/GLUT.
